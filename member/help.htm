사진 올리기 도움말


1. 사진은 120픽셀 x 150픽셀로 올리셔야 합니다. 크기가 약간 차이가 나도 상관 없지만 여러분 얼굴이 찌그러져 보이기를 원하지는 않으시겠죠?


2. Adobe Photoshop을 이용하여 사진 크기를 조정하는 방법을 알아 보도록 하죠!! (편의상 경어를 생략하겠습니다.)


1) 원본 사진 이미지를 준비하고 포토샵에서 이미지를 연다. File > Open..

2) 왼쪽 도구모음에서 Marquee툴을 선택한다.


3) 오른쪽의 Option Box에서 Fixed size를 선택하고 Width : 120, Height : 150을 선택하고 이미지를 클릭한다.


4) 이제 120 x 150 크기의 그림이 선택되었을 것이다. 선택 박스가 너무 크거나 작은 경우는 Image > Image Size.. 메뉴에서 원본이미지를 적당한 크기로 조정을 한다. (원하는 크기가 될 때까지..)

5) 이제 모든 준비가 끝났다. Image > Crop을 선택하면 선택한 부분의 이미지만이 남아 있을 것이다.

6) File > Save As.. 에서 jpg 형식으로 저장 형식을 바꾼후 저장한다.
